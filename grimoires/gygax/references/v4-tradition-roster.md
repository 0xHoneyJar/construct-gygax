# v4 Tradition Roster

20 traditions total. Each gets a lore heuristic YAML file with curated design wisdom and known anti-patterns.

## Existing (TTRPG)

| Tradition | Design DNA | Canonical Examples |
|-----------|-----------|-------------------|
| d20 | Bounded accuracy, class/level, action economy, CR math | D&D, Pathfinder |
| pbta | Fiction-first, 2d6+stat, moves as tripwires, partial success | Apocalypse World, Masks |
| fitd | Position/effect, dice pools, stress/trauma, clocks | Blades in the Dark |
| osr | Procedures, reaction tables, resource depletion, player skill > character skill | Cairn, Old School Essentials |
| cepheus | 2d6+mod, lifepath generation, effect system, skill-based | Traveller, Cepheus Engine |
| freeform | Minimal mechanics, narrative authority, prompts, facilitation | Wanderhome, For the Queen |
| custom | Novel systems that don't fit established traditions | Any hybrid or experimental design |

## New (v4)

| Tradition | Design DNA | Canonical Examples |
|-----------|-----------|-------------------|
| autobattler | Draft economy, synergy tables, positioning grids, automated resolution, lobby meta, item economy | TFT, Super Auto Pets, Storybook Brawl |

## Key Design Differences Per Tradition

What augury should look for differently:

| Tradition | Primary Analysis Concern | Secondary |
|-----------|------------------------|-----------|
| autobattler | Synergy balance, draft fairness, positioning value | Meta diversity, item economy |
| roguelike | Build path viability, run variance, difficulty curve | Synergy discovery rate, permadeath fairness |
| deckbuilder | Card economy (buy value vs deck dilution), engine ceiling | Thinning efficiency, turn tempo |
| ccg | Mana curve math, card advantage, archetype matchup spread | Meta polarization, design space consumption |
| 4x | Victory path balance, snowball detection, mid-game pacing | Tech tree dead ends, diplomatic exploit |
| tactics | Map design fairness, unit composition balance, action economy | Permadeath recovery, difficulty spikes |
| eurogame | Resource conversion efficiency, engine ceiling, point paths | Player interaction level, runaway leader |
| social-deduction | Role balance, information flow fairness, game length | Elimination timing, kingmaking |
| cooperative | Difficulty curve, alpha player surface, role necessity | Scaling with player count, loss feel |
| idle | Prestige curve shape, automation pacing, offline value | Number legibility, decision density |
| extraction | Risk/reward calibration, loss aversion tuning, loot table fairness | Information asymmetry balance, insurance value |
| looter | Loot distribution, build diversity, endgame engagement | Power creep rate, crafting vs drop value |
| immersive-sim | Solution path diversity, systemic interaction depth, ability gate pacing | Environmental readability, player expression breadth |
