# Cabal

Simulate playtesting sessions with synthetic player archetypes to stress-test TTRPG mechanics. This skill models how different player types (optimizers, narrativists, rules lawyers, casual players) would interact with game mechanics, surfacing exploits, edge cases, and feel-bad moments.

- Focuses on simulated player behavior, exploit discovery, and interaction dynamics
