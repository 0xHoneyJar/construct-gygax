# Augury

Perform balance analysis on TTRPG mechanics using quantitative methods (DPR calculations, probability distributions, encounter math) and qualitative heuristics. This skill evaluates whether game elements are appropriately balanced relative to each other and to established benchmarks.

- Focuses on numerical analysis, probability modeling, and comparative evaluation
