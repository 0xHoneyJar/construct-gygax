# Homebrew

Design new TTRPG mechanics or refine existing ones through structured iteration. This skill takes a mechanical concept or design prompt and produces well-defined game mechanics with consideration for clarity, usability, and integration with the target system.

## Trigger

/homebrew

## Workflow

Placeholder -- full workflow defined in a later sprint.

## Boundaries

- Does NOT ingest or parse existing rulebooks (use /attune for that)
- Does NOT perform quantitative balance analysis (use /augury for that)
- Does NOT simulate player interactions (use /cabal for that)
- Focuses on mechanical design, wording clarity, and system coherence
